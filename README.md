# pgp-pcie-apps

[DOE Code](https://www.osti.gov/doecode/biblio/75501)

`pgp-pcie-apps` is firmware and software for moving data between PGP
optical links (commonly connected to detectors) and a PCIe DMA engine,
built on top of `axi-pcie-core` and `surf` for use with SLAC PGP-class
accelerator cards.

See the documentation at https://slaclab.github.io/pgp-pcie-apps/.
